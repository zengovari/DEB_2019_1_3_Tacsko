Rendszerterv
Időjárás előrejelző program

I. A rendszer: Egy magyar nyelvű időjárás előrejelző program, amely egy modern és könnyen hasznáható felhasználói felületet használ, hogy ídős emberek is könnyen boldoguljanak vele.

II. A rendszer célja: Egy nagyon egyszerűen használható, a lehető legátláthatóbb időjárás előrejelzés biztosítása a jelenlegi helyszín, illetve akár egy a felhasználó által megadott helyszín alapján idős emberek számára, hogy ők saját
önerejükből képesek legyenek tájékozodni az általuk kiválasztott helyiség ídőjáráselőrejelzésével kapcsolatba, hogy az öregotthon lakói rendelkezzenek egy biztos időjáráselőrejelzési forrással, melyet bármikor kényelmesen megtudnak názni
anélkül, hogy bárkinek is problémat okozzanak vele.

III. Terv: A megbizóink kéréseiből a programot könnyedén magyar nyelvre tudjuk szabni,
viszont a települések neveit csak a hivatalos elnevézeseik szerint tudjuk csak ábrázolni, de figyelembe véve, hogy az idősotthon lakóit
leginkább a helyi időjárásviszonyok foglalkoztatják, így kétlem, hogy nagy kellemetlenség lenne belöle.
Az öt napos ídőjáráselőrejelzés sem fog gondot okozni, szándékosan egy programkönyvtárat fogunk használni amivel öt napra előre lehet tudni az ídőjárási körülményeket, hogy a lakóknak ne kelljen minden egyes nap aggódniuk amiatt, hogy
a könyvtári számitógép el lesz foglalva akkor amikor megszeretnék tudni milyen ídőjárás fog bekövetkezni aznap, a programunkkal képesek lesznek több napi ídőjárási viszonyokról tájékozodni.
A felhasználói felületet úgy fogjuk megtervezni, hogy az látási problémákkal szenvedőknek is kényelmes legyen, hogy ne rontsák tovább szemeiket hunyoritással, illetve a képernyőhöz ne kelljen közelítsenek ahhoz, hogy lássák milyen részletek vannak az ídőjáráselőrejelzésben.
A felület felhasználó barát lesz, úgy értve, hogy minden lehetséges opció az alkalmzás megnyitásakór a felhasználó szeme előtt lesz. Nem lesznek benne alopciók, így bámilyen szakértelemmel is rendelkezne a felhasználó könnyedén képes lesz
eligazodni a program felületén.  
Igyekszünk eleget tenni a kéréseknek, így a programot úgy tervezzük meg, hogy a lehető legkevesebb müvelettel meg lehessen nézni az ídőjáráselőrejelzést. A felhasználónak csupán a helyiséget kell kiválasztania és a program máris ki fogja
mutani milyen ídőjárási viszonyok vannak most illetve fognak bekövetkezni a következő 5 napban, kételljük, hogy így valamelyik lakósnak probléma merülne fel a program használatakor.
A képernyővédelmes kérelmet figyelembe véve, nem vagyunk biztosak abban, hogy képesek leszünk ilyen rövid határídővel ezt az opciót megvalósítani, így nem fog szerepelni a munkatervben, nem ígérhetünk valamit amibe nem vagyunk biztosak.
A felhasználói felületünkön a felhasználónknak lehetősége lesz kiválasztani melyik települéről szeretne ídőjáráselőrejelzést kiválasztani. Amikor ezt kiválasztja akkor az OpenWeather API lekéri a településsel kapcsolatos ídőjárási adatokat az API oldaláról és ezeket lementi egy JSON fájlba. Tehát az első lépés magának a JSON fájlnak a beolvasása kell hogy legyen. A terv az, hogy minden menüpont a program megnyitásával elérhető, könnyen megtalálható legyen, minél nagyobb ikonok használata, hogy a látási problémákkal szenvedők is könnyen be tudhassák azonosítani a képernyőn található informáciőkat. Ahogy a minta képen is látható, a bal felső sarokban helyezkedik el magának a jelenlegi vagy a felhasználó által választott helyszínnek a neve, azalatt pedig a jelenlegi időjárás részletesen (jelenlegi időjárás az OpenWeather API által szolgáltatott ikonnal szemléltetve, hőmérséklet, szélsebesség, szélirány, borultság százalékban megadva, nyomás, páratartalom). A felhasználónak a grafikus felület jobb felső oldalán van lehetősége más helyszín választására. A jelenlegi időjárás alatt, 5 egymástól elkülönített kis ablakaban jelenik meg az 5 napos előrejelzés röviden (időjárás ikonnal szemléltetve, hőmérséklet, szélsebesség, szélirány), melyekkel az egérrel interakcióba lehet lépni, kattintás után pedig a megfelelő nap részletes előrejelzését láthatja a felhasználó.   


IV. Időpont: A programnak várhatóan 2019. szeptember 30. -ra (hétfő) kell készen lennie.

V. Erőforrások: A programozási nyelv a Java 11 lesz. A választás azért esett a Java programozási nyelvre, mivel a felhasználó felület megvalósításához JavaFX 12-t használunk, amivel rendkívül gyorsan és egyszerűen tudunk grafikus felülettel rendlező alkalmazásokat készíteni köszönhetően a screen buildernek, illetve számos olyan opció van benne mely megkönnyiti a fejlesztési folyamatokat.
Az OpenWeather API (https://openweathermap.org/), aminek fő választási oka az volt, hogy képes nem csak a jelenlegi időjárási körülményeket megadni hanem képes több napos előrejelzést is nyújtani, ingyenes licencét használva hozzáférhetünk a megadott helyszín (név, ID, koordináta, irányítószám formájában megadható) jelenlegi időjárására, illetve 5 napos előlejelzésére. Az API válaszul egy JSON fájlt küld vissza, melyet a Google Gson (nyílt forráskodú Java könyvtár mely Java objektumot képes JSON-é, illetve JSON fájlt képes Java objektummá alakítani), segítségével olvasunk be, így a feldolgozásuk sokkal egyszerűbb és gyorsabb. Mivel Google Gsont használunk, így szükséges, hogy a JSON fájlba az olyan elemek amelyek több paraméterrel rendelkeznek Java osztályként jelenjenek meg a kódunkban, így rendkívül könnyen kezelhetővé, hozzáférhetővé válik minden API által szolgáltatott információ.

A jelenlegi időjáráshoz szükséges request formája: https://samples.openweathermap.org/data/2.5/weather?q=HELYSZÍN&appid=API_KEY, az 5 napos előrejelzéshez szükséges request formája: https://samples.openweathermap.org/data/2.5/forecast?q=HELYSZÍN&appid=API_KEY, az API_KEY-t az OpenWeather API biztosítja. A JSON fájlra a mellékletben található egy példa (example.json).
Az API által küldött JSON fájl rengeteg hasznos információt szolgál számunkra, melyek közül az alábbiakat használjuk: 
- Maximum hőmérséklet, minimum hőmérséklet, jelenlegi hőmérséklet
- Nyomás
- Páratartalom
- Szélsebesség, szélirány
- Felhősség (százalékban megadva)
- Naplemente, napnyugta
- 5 napos előrejelzés
- Ikon ami a jelenlegi időjárást szemlélteti
