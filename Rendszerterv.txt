Rendszerterv
Időjárás előrejelző program

I. A rendszer: Egy magyar nyelvű időjárás előrejelző program.

II. A rendszer célja: Egy nagyon egyszerűen használható, a lehető legátlátóbb időjárás előrejelzés biztosítása a jelenlegi helyszín, illetve akár egy a felhasználó által megadott helyszín alapján idős emberek számára.

III. Terv: Az első lépés magának a JSON fájlnak a beolvasása kell hogy legyen. A terv az, hogy minden menüpont a program megnyitásával elérhető, könnyen megtalálható legyen. Mint a minta képen is látható, a bal felső sarokba helyezkedik el magának a jelenlegi vagy a felhasználó által választott helyszínnek a neve, azalatt pedig a jelenlegi időjárás részletesen, (jelenlegi időjárás az OpenWeather API által szolgáltatott ikonnal szemléltetve, hőmérséklet, szélsebesség, szélirány, borultság százalékban megadva, nyomás, páratartalom). A felhasználónak a grafikus felület jobb felső oldalán van lehetősége más helyszín választására. A jelenlegi időjárás alatt 5 ablakban jelenne meg az 5 napos előrejelzés röviden (időjárás ikonnal szemléltetve, hőmérséklet, szélsebesség, szélirány) külön ablakokba, melyek kattinhatóak, kattintás után a megfelelő nap részletes előrejelzését láthatja a felhasználó.   

IV. Időpont: A programnak várhatóan szeptember 30. -ra (hétfő) kell készen lennie.

V. Erőforrások: A programozási nyelv a Java 11 lesz, a felhasználó felület megvalósításához JavaFX 12-t használunk.
Az OpenWeather API (https://openweathermap.org/) ingyenes licencét használva hozzáférhetünk a megadott helyszín (név, ID, koordináta, irányítószám formájában megadható) jelenlegi időjárására, illetve 5 napos előlejelzésére. Az API válaszul egy JSON fájlt küld vissza, melyet a Google Gson (nyílt forráskodú Java könyvtár mely Java objektumot képes JSON-é, illetve JSON fájlt képes Java objektummá alakítani), segítségével olvasunk be.
A jelenlegi időjáráshoz szükséges request formája: https://samples.openweathermap.org/data/2.5/weather?q=HELYSZÍN&appid=API_KEY, az 5 napos előrejelzéshez szükséges request formája: https://samples.openweathermap.org/data/2.5/forecast?q=HELYSZÍN&appid=API_KEY, az API_KEY-t az OpenWeather API biztosítja. A JSON fájlra a mellékletben található egy példa (example.json).
Mivel Google Gsont használunk, így szükséges, hogy a JSON fájlba az olyan elemek amelyek több paraméterrel rendelkeznek Java osztályként jelenjenek meg a kódunkban, így a feldolgozásuk sokkal egyszerűbb és gyorsabb. (melléklet AdatUML)
Az API által küldött JSON fájl rengeteg hasznos információt szolgál számunkra, melyek közül az alábbiakat használjuk: 
- Maximum hőmérséklet, minimum hőmérséklet, jelenlegi hőmérséklet
- Nyomás
- Páratartalom
- Szélsebesség, szélirány
- Felhősség (százalékban megadva)
- Naplemente, napnyugta
- 5 napos előrejelzés
